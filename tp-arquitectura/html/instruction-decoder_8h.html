Documentación de las enumeraciones

enum _instructionType

Describe todos los tipos de instruccion posibles.

Valores de enumeraciones
ASSIGN 

Asigna el valor de una expresion a una variable.

INPUT 

Lee un valor entero de stdin y lo escribe a una variable.

OUTPUT 

Escribe el valor de una expresion en la pantalla.

END 

Termina el programa.

JUMP 

Cambia el punto de ejecucion.

LABEL 

Declara una etiqueta, lugar al cual se puede cambiar el punto de ejecucion.

CONDITIONAL_JUMP 

Cambia condicionalmente el punto de ejecucion.
